03 / 13
활성화 함수 | Sigmoid 함수

핵심 특성

  • 비선형성 — S자 형태의 곡선으로 휘어짐
  • 확률 변환 — 어떤 실수도 0~1 사이로 압축
  • 이진 분류 문제에 핵심적으로 활용
  • 출력 0.5 이상 → 클래스 1 / 미만 → 클래스 0
σ(x) = 1 / (1 + e−x)

한계점

  • ① 기울기 소실 — x가 커질수록 기울기→0
  • ② 역전파 방해 — 깊은 층까지 오차 전달 불가
  • ③ 학습 불가 — 깊은 신경망에서 오차 소멸
04 / 13
활성화 함수 | ReLU 함수 — Sigmoid의 한계 극복
ReLU(x) = max(0, x)

장점

  • 간단한 규칙 — 음수는 0, 양수는 그대로
  • 기울기 유지 — 양수 구간 기울기 = 1 (소실 없음)
  • 연산 효율 — 지수 연산 없이 최고 효율
  • 깊은 네트워크에서도 오차 전달 가능

주의 — Dying ReLU

음수 입력 시 노드가 완전히 비활성화되어
학습이 중단되는 현상 발생 가능

Sigmoid
기울기 소실
깊은 층 학습 ✗
지수 연산 필요
ReLU
기울기 유지
깊은 층 학습 ✓
연산 효율 최상

은닉층 — ReLU 사용

음수 신호를 0으로 차단하여 불필요한 뉴런 비활성화.
깊은 네트워크에서도 기울기 소실 없이 학습 가능.

출력층 — Sigmoid 사용

결과를 0~1 사이 확률값으로 압축.
이진 분류에서 최종 예측값을 확률로 해석하기 위해 사용.

08 / 13
오차 역전파 | Backpropagation — 연쇄법칙과 가중치 업데이트
① 순전파
Output = X · W
예측값 생성 → Loss 계산
② 오차 역전달 (dX)
dX = dY · Wᵀ
이전 층으로 오차 신호 전달
③ 가중치 업데이트 (dW)
dW = Xᵀ · dY
현재 층의 가중치 W 수정
결과물 역할 수식 비유
dX 신호 전달 dX = dY · Wᵀ "뒷 층에 이만큼 고쳐달래요!"
dW 성능 개선 dW = Xᵀ · dY "내 가중치를 요만큼 깎아야겠군."
W_new 가중치 갱신 W_new = W_old − α · dW α = 학습률 (보폭의 크기)

연쇄법칙 (Chain Rule)

∂E/∂w = (∂E/∂a) · (∂a/∂z) · (∂z/∂w)
각 레이어의 변화율을 순서대로 곱하여
입력 가중치까지 오차를 역방향으로 전달.

전치 행렬 (Wᵀ) 의 역할

역전파 시 오차를 입력 데이터의 차원에
맞춰 전달하기 위해 전치 행렬 사용.
레이어를 아무리 깊게 쌓아도 동일 원리 적용.
